PREGUNTAS GUÍAS.
Ejercicio 2: Flexbox

    ¿Por qué cambia el posicionamiento de las cajas internas?
    El posicionamiento cambia por la propiedad del display: flex, que hace que los elementos hijos se adapten al espacio disponible.

    ¿Se puede cambiar el orden de los elementos solo con CSS?
    Sí, es posible. Flexbox permite reorganizar elementos sin modificar el HTML o JavaScript.

    ¿Qué hacen las propiedades?
    
        flex-wrap: wrap: Permite que los elementos se ajusten y pasen a una nueva línea si no caben en la misma.

        width: Establece el ancho inicial de los elementos flexibles.

Ejercicio 3: Grid Avanzado

    ¿Puedes diseñar toda tu web usando GRID?
    Sí. CSS Grid es una herramienta ideal para diseñar maquetaciones completas, ya que fue creada específicamente para resolver este tipo de problemas de diseño web.

    ¿Por qué los cambios se dan con tan pocas líneas de CSS?
    Los cambios son tan eficientes porque display: grid y grid-template-areas te permiten definir una cuadrícula visualmente en el CSS. Esto simplifica la maquetación y facilita la reorganización del diseño con muy poco código.
